# Build Plan

This file outlines the sequential order in which we will build the HR & Payroll Management System frontend. Every unit must be completed and verified before moving to the next.

## Units

1. **Unit 01: Project Setup**
   - Initialize Next.js app, configure Tailwind, and set up shadcn/ui.
2. **Unit 02: API & Auth Context**
   - Create Axios interceptors and global Zustand/React Context for user session state.
3. **Unit 03: Base Dashboard Layout**
   - Build the responsive Sidebar and Header shells for Admin, Manager, and Employee views.
4. **Unit 04: Authentication UI**
   - Build the Login page and Next.js middleware for role-based route protection.
5. **Unit 05: Employee Management (Admin)**
   - Build the CRUD interface for creating, viewing, and editing employee records.
6. **Unit 06: Attendance Management**
   - Build employee check-in/out interface and manager/admin attendance view.
7. **Unit 07: Leave Management**
   - Build leave request form for employees and approval workflow for managers.
8. **Unit 08: Salary Structure (Admin)**
   - Build UI for defining salary components and assigning structures to employees.
9. **Unit 09: Payroll Processing**
   - Build the payroll run trigger for admins and payslip viewer for employees.
10. **Unit 10: Loan Management**
    - Build loan request form and admin approval interface.
11. **Unit 11: Reporting & Analytics**
    - Build charts and data summaries for payroll costs and attendance trends.
